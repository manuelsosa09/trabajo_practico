# InstrumentHub

## Descripción del Proyecto

InstrumentHub es una aplicación desarrollada en Python utilizando Flet para la gestión y visualización de instrumentos musicales. La aplicación permite explorar instrumentos, visualizar información detallada, gestionar favoritos y manejar usuarios mediante autenticación.

## Cómo Ejecutar el Proyecto

### 1. Clonar el repositorio

```bash
git clone https://github.com/manuelsosa09/trabajo_practico.git
```

### 2. Ingresar a la carpeta del proyecto

```bash
cd trabajo_practico
```

### 3. Instalar las dependencias necesarias

```bash
pip install flet
```

### 4. Ejecutar la aplicación

```bash
python main.py
```
